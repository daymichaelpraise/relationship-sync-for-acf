# v1.1.0 — original

The initial, site-specific plugin ("NEC Relationship Sync"). Bidirectional sync was hardcoded between the `nec-team` post type and the `nec-ministry` / `nec-service` post types via two fixed ACF fields. Included a Tools → NEC Sync Debug page.

This snapshot is preserved for history. Note: v1.1.0 wrote relationship values with `update_post_meta` without the ACF field-key reference (fixed in v1.2.0).
